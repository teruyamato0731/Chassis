Chassisを用いて手動制御、自己位置推定、自動制御を行う方法を示す。 下記の例ではMotor.hを使用している。

手動制御

frameモジュールを用いる。 例はOmniを制御するものである。

インスタンスの生成

Omni<3> omni{[](const float (&pwm)[3]) {
for(int i = 0; i < 3; ++i) {
motors[i] = pwm[i];
}
}};

移動速度を渡してモータ出力を計算しモータにセットする。

Velocity vel = {0, 0.5, 0};
omni.move(vel);

4輪独立ステアリング

インスタンスの生成

SteerDrive<4> steer{[](const std::complex<float> (&cmp)[4]) {
for(int i = 0; i < 4; ++i) {
motors[i] = abs(cmp[i]);
servo[i] = arg(cmp[i]);
}
}};

出力

Velocity vel = {0, 0.5, 0};
steer.move(vel);
